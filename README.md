# Mrinmoy Portfolio

This repository contains a static personal portfolio website for Mrinmoy Chakraborty.

## Deploy to GitHub Pages

1. Push this code to the `Mrinmoy-portfolio` repository on GitHub.
2. In GitHub, open **Settings** → **Pages**.
3. Under **Build and deployment**:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main` (or your default branch), `/root`
4. Save and wait ~1-2 minutes.
5. Your site will be available at:
   `https://<your-github-username>.github.io/Mrinmoy-portfolio/`

## Local preview

Open `index.html` in a browser, or run a simple local server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
